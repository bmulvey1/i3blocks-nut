# i3blocks-nut

i3blocks script for displaying UPS status

## Dependencies

https://pypi.org/project/PyNUTClient/

## Config

Edit the script and set these variables to the appropriate values for your config:
```python
HOST = "127.0.0.1"
PORT = 3493
USER = "username"
PASSWORD = "password"
UPSNAME = "upsname"
```
